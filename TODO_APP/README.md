# ⚓ The Grand Line Log • One Piece Todo App

A premium, cinematic Todo application themed after the legendary **Straw Hat Pirates**. Designed with a "Grand Line Noir" aesthetic, this app turns your daily tasks into epic pirate quests.

## 🏴‍☠️ Features

- **Grand Line Noir UI**: High-fidelity design system featuring dark mahogany textures, weathered parchment, and gold-leaf accents.
- **Straw Hat Crew Dashboard**: Interactive header featuring all 10 crew members with unique "Haki Aura" hover effects.
- **Wanted Poster Quests**: Every task is rendered as a diegetic Wanted Poster with irregular torn edges and random Beli rewards.
- **Cinematic Animations**: 
    - **Desk Slam**: Posters slam onto the desk with physical impact.
    - **Wax Seal**: Claiming a bounty triggers a high-impact stamp press.
    - **Haki Pulses**: Crew icons shimmer with their signature colors.
- **Professional & Responsive**: Fully optimized for both desktop and mobile devices.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure for a clean logbook experience.
- **Vanilla CSS3**: Advanced glassmorphism, clip-paths, and GPU-accelerated keyframe animations.
- **JavaScript**: Robust CRUD logic with "Quest Lead" assignments and dynamic reward generation.

## 🚀 How to Set Sail

1. Download or clone this repository.
2. Open `todo.html` in any modern web browser.
3. Start scribing your quests, Captain!
